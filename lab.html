Quiz Forense

1. Qual è la posizione del file sospetto individuato nei log di Windows?

Controlla la directory temporanea dell’utente in AppData\Local.

2. Quale comando di Volatility può essere usato per elencare i processi attivi?

È simile al comando “pslist”.

3. Qual è l’indirizzo IP del server remoto usato per l’esfiltrazione DNS?

Verifica nei log DNS o nel pcap filtrando per protocollo UDP.
